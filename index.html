EtherDelta Extension Bookmarklets

by Steve Klebanoff

All extensions are simple javascript which can be inspected by viewing the link source. This guarantees there are no exploits or malicious code executed.

How to setup:

  • Make sure your bookmarks bar is showing your browser.
  • Drag the appropriate bookmarket (i.e. "Set Title") link into your bookmarks bar.
  • Once you have navigated to an EtherDelta page, click the bookmark and watch the functionality work.
